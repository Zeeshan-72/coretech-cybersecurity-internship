# Methodology

The assessment followed the OWASP Web Security Testing Guide.

## Testing Phases

1. Information Gathering
2. Configuration Testing
3. Authentication Testing
4. Authorization Testing
5. Input Validation Testing
6. Session Management Testing
7. Reporting and Findings

## Tools Used

- Kali Linux
- OWASP WSTG
- SecurityHeaders.com
- SSL Labs
- GitHub

The testing was conducted using passive and non-destructive techniques.
