# Appendix B вЂ“ Assessment Scope

The security assessment focused on evaluating the security posture of the selected web application using the OWASP Web Security Testing Guide (WSTG). The assessment was performed for educational purposes as part of the CoreTech Cybersecurity Internship.

The following testing areas were included:

- Information Gathering
- Configuration Testing
- Authentication Testing
- Authorization Testing
- Input Validation Testing
- Session Management Testing
- Security Header Analysis

# Appendix D вЂ“ Risk Rating Scale

| Severity | Description |
|----------|-------------|
